# PRD: Basic Learning Track

## Capability: AI-Powered Administrative Foundations

**Description:** This capability enables university administrative staff to set up an AI-powered work environment and understand the fundamental concepts of interacting with GitHub Copilot for basic document creation and assistance.

### Feature 1: Environment Setup
- **Description:** Establishes the necessary software and tools, including VS Code and the GitHub Copilot extension.
- **User Story:** As an administrator, I want to install and configure my digital workspace so that I can start using AI tools for my tasks.

### Feature 2: Core Concepts
- **Description:** Differentiates between the two primary modes of Copilot interaction: interactive Chat and inline Autocomplete (Ghost Text).
- **User Story:** As an administrator, I want to understand when to use Chat versus Autocomplete so that I can choose the most efficient method for my current task.

### Feature 3: Practical Application (Workbook)
- **Description:** Guides the user through a hands-on exercise of drafting a formal email in English, applying both Chat and Autocomplete functions.
- **User Story:** As an administrator, I want to complete a practical exercise so that I can solidify my understanding of Copilot's basic functions.
